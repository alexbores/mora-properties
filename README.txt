Folder structure:

/templates - For specific page layouts or templates.
/components - For reusable PHP components or partials.
/assets - Dedicated to media files.
/fonts - To store custom font files.
/styles - For all theme CSS files.
/scripts - For theme JavaScript files.
/base - For foundational styles, scripts and php that are not editable.
/vendor - For external libraries.
/handlers - For AJAX and form handling scripts.
/inc - For core theme functionality and helper PHP scripts that are editable.
